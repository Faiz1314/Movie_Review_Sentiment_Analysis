# Movie Review Sentiment Analysis 🎬

This project performs **Sentiment Analysis on Movie Reviews** using Natural Language Processing (NLP) and Machine Learning techniques.

The goal of this project is to classify movie reviews as **Positive** or **Negative** based on the text of the review.

---

## Project Overview

Sentiment Analysis is a common NLP task used to determine the emotional tone behind a piece of text. In this project, we analyze movie reviews and build machine learning models to predict whether a review expresses a positive or negative sentiment.

---

## Features

- Text preprocessing and cleaning
- Feature extraction using NLP techniques
- Training multiple machine learning models
- Evaluating model performance
- Testing predictions on new reviews

---

## Technologies Used

- Python
- Natural Language Processing (NLP)
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

---

## Project Workflow

1. **Text Preprocessing**
   - Lowercasing
   - Removing punctuation
   - Removing stopwords
   - Tokenization

2. **Feature Extraction**
   - Bag of Words
   - TF-IDF Vectorization

3. **Model Training**
   - Logistic Regression
   - Naive Bayes
   - Other classification models

4. **Model Evaluation**
   - Accuracy
   - Confusion Matrix
   - Model comparison

5. **Prediction**
   - Testing sentiment on unseen movie reviews
